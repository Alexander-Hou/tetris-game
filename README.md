# C语言俄罗斯方块

一个在Windows终端中运行的俄罗斯方块游戏。（纯C语言，无外部游戏引擎，字符界面，无图形库）


## 游戏功能
- 7种经典方块（I、J、L、O、S、T、Z）
- 旋转、移动、碰撞检测
- Hold功能（暂存方块）
- Next预览（显示下一个）
- 分数记录和等级系统
- 彩色游戏界面

## 操作按键

| 功能 | 按键 |
|------|------|
| 左移 | ← 或 A |
| 右移 | → 或 D |
| 旋转 | ↑ 或 W |
| 快速下落 | ↓ 或 S |
| 暂存方块 | 空格键 |
| 暂停/继续 | P |
| 重新开始 | R |
| 退出游戏 | Q |

## 文件说明
- `main.c` - 主程序入口
- `board.c` - 界面显示
- `block.c` - 方块操作
- `logic.c` - 游戏逻辑
- `score.c` - 分数管理
- `tetris.h` - 头文件

## 游戏目标
- 消除行数获得分数
- 升级速度加快
- 挑战最高分记录
